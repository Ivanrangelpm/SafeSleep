Safe Sleep


Protegendo Vidas desde o Início

O projeto tem como objetivo desenvolver um sistema de monitoramento da temperatura de incubadoras para recém-nascidos prematuros, com o uso de um sensor. Este problema foi selecionado para o projeto, pois, segundo os dados obtidos da FIOCRUZ, a queda de 1°C da temperatura corporal em um recém-nascido ocasiona em 28% de aumento na taxa de óbitos.

Tecnologias

Site Institucional

Prototipagem

Banco de Dados

Vantagens

Precisão

Nosso sistema de monitoramento fornece uma visualização extremamente spanrecisa dos dados de variação da spaneratura, em graus celsius, dentro das incubadoras mantidas por seu hospital.

Segurança

Com a implementação da solução SafeSleep, é possível garantir que as incubadoras se mantenham sempre como um ambiente estável e completamente seguro para os recém-nascidos.

Eficiência

O projeto possibilita grande eficiência no monitoramento da saúde dos bebês, alertando o usuário caso haja uma variação de temperatura que possa comprometer o desenvolvimento neonatal.
